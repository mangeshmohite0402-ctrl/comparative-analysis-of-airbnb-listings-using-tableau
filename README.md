# 🏡 Comparative Analysis of Airbnb Listings Using Tableau

# 📄 Description:
Analyzed Airbnb public listings data from Chicago and New Orleans, cleaned and transformed it into a dataset (airbnb-eda), and built an interactive Tableau dashboard. The dashboard compares city-level trends in pricing, availability, property types, and host performance.

# 🧩 Key Tasks

Cleaned raw Airbnb dataset and created airbnb-eda file

Standardized neighborhood, room types, and categorical fields

Removed duplicates & handled missing values

Built multiple Tableau visualizations (heatmaps, tree maps, bar charts, KPIs)

Created a comparative dashboard for Chicago vs New Orleans

# 📊 Insights

* **Market Availability:**

  * Chicago: 1,302,368 listings
  * New Orleans: 926,431 listings

* **Revenue Drivers:**

  * Entire Homes/Apartments contributed the highest revenue (**2,000,061**)
  * Private Rooms: **204,144**
  * Hotel Rooms & Shared Rooms: negligible

* **Top Hosts:**

  * Jane: **28,141**
  * Michael: **24,389**
  * Blueground: **23,664**

* **Stay Duration:**

  * Most common: **1 night with 1 bedroom (5,616 listings)**
  * Other frequent: 2 nights (2,381), 0 nights (1,825)

* **Capacity Trends:**

  * Majority properties: **2–3 bedrooms, 1–2 bathrooms**
  * Total Beds: **23,640**
  * Total Bedrooms: **22,918**
  * Total Bathrooms: **18,562**

* **Customer Experience:**

  * Average Rating: **4.38**
